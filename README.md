# IT516 React Website Project

This project was developed for IT516 Web Information Systems at Southern States University.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- React
- Next.js
- Vercel

## Features

- Responsive layout
- Dark mode toggle
- Contact form validation
- React component architecture

## Running the Project Locally

1. Open the project folder in Visual Studio Code
2. Open the terminal
3. Run:

npm install

4. Start the development server:

npm run dev

5. Open:

http://localhost:3000

## Deployment

The project is deployed using Vercel.
